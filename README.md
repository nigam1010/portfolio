# 🏎️ KSA Racing Portfolio

Welcome to the **KSA Racing Portfolio**, a high-performance, F1-inspired personal portfolio designed to showcase premium software engineering capabilities through an immersive, cinematic web experience.

## ✨ Highlights
- **Engine Start Cinematic Intro:** A highly-optimized, 60fps SVG-masked engine blast animation that seamlessly unmounts into the main experience.
- **Canvas Flowing Light Trails:** Dynamic speed-line backgrounds powered by the HTML5 Canvas API in the Hero section.
- **Orbital Skills Matrix:** Interactive, spinning 3D-styled UI components to visualize my tech stack.
- **Pitch-Ready Project Showcase:** A dynamic rendering engine that transforms project cards into full-page startup-style pitch decks containing image galleries, problem statements, and real-world architectures.

## 🛠️ Built With...
- **React.js** (Functional Components, Hooks)
- **Vite** (Next-generation frontend tooling)
- **Framer Motion** (Production-ready physics animations)
- **Vanilla CSS3** (Custom variables, complex grid layouts, absolute tracking)
- **Canvas API** (Performant particle rendering)

## 📂 The Showcase

### 1. MotoMind - Premium Car Configurator
An AI-extendable luxury car customization sandbox. Designed to deliver a premium digital showroom experience bridging the gap between static configurators and end-to-end purchasing flows.
*(Stack: React, Vite, Node, Express, PayPal SDK)*

### 2. CanovaCRM
A full-stack, role-based Customer Relationship Management tool solving real-world workflow design friction by catering unified routing to Admins and Mobile-first Field Employees.
*(Stack: React, Node.js, Express, MongoDB)*

### 3. Multilingual IoT Voice Controller
An AI-powered IoT system entirely driven by voice commands running a simultaneous pipeline for English, Hindi, and Kannada, bypassing standard touch-interfaces using LLMs and MQTT orchestration.
*(Stack: Python, Raspberry Pi, MQTT, LLM)*

### 4. Telematics Behavior Classification
Published ML research indexed on Google Scholar classifying aggressive, normal, and drowsy driver behavior metrics using Random Forests and Deep Learning.
*(Stack: Python, Scikit-learn, TensorFlow, LSTM)*

## 🚀 Running Locally

```bash
# Clone the repository
git clone <your-repo-url>

# Navigate into the project directory
cd racing-portfolio

# Install dependencies
npm install

# Start the Vite development server
npm run dev
```
